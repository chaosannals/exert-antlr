# JVM

## 类型描述

- B 基本数据类型byte
- C 基本数据类型char
- D 基本数据类型double
- F 基本数据类型float
- I 基本数据类型int
- J 基本数据类型long
- S 基本数据类型short
- Z 基本数据类型boolean
- V 基本数据类型void
- L; 对象类型（L开头，类路径，;结尾），如Ljava/lang/Object;
- [ 数组类型（[开头，描述符），如[Ljava/lang/Object; 和 [I
- () 函数类型（括号里面是参数描述，返回类型结尾），如 ()V 和 (I[Ljava/lang/String;)Z
